# Eye Control AI

A 100% buildable, production-ready Android application.
Built using **Kotlin, Jetpack Compose, CameraX, and Android Speech Recognition**.

## Architecture & Frameworks
- **UI:** Jetpack Compose (Material Design 3)
- **Architecture:** MVVM
- **Database/Persistence:** Jetpack DataStore Preferences
- **Hardware Integration:** CameraX for Front Camera Eye Scanning
- **Accessibility:** Android Accessibility Service implementation
- **Voice Typing:** Android SpeechRecognizer engine

## Build Instructions
1. Open this project in **Android Studio (Hedgehog or newer)**.
2. Ensure you have **JDK 21** configured.
3. Build and install using the play button, or run './gradlew assembleRelease' to compile the release APK.
